Conditional Statements

Conditional statements are used to perform different actions based on different conditions.

Syntax

if (condition) {
  // code if true
} else if (anotherCondition) {
  // code if second is true
} else {
  // code if all are false
}

Practical: Even / Odd Checker

Revision:
  • if – executes block when condition is true.
  • else if – checks another condition.
  • else – fallback when all previous conditions are false.
