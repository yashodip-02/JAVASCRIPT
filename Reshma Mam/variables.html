Variables in JavaScript

A variable is a container used to store data values in JavaScript.

Declaration Keywords

Keyword Scope Redeclare? Reassign? Notes
var Function / Global Yes Yes Old style, hoisted
let Block No Yes Modern, preferred for changing values
const Block No No For constants (cannot reassign)
