Introduction to JavaScript

JavaScript is a programming language used to create interactive websites. It allows you to add dynamic content such as sliders, popups, and calculations.

History of JavaScript

JavaScript was created in 1995 by Brendan Eich at Netscape. It was initially called LiveScript and later renamed JavaScript.

Client-Side vs Server-Side

  • Client-Side JS: Runs inside browser. Used for UI and interactions.
  • Server-Side JS: Runs on server using Node.js. Used for backend logic.

Variables (var, let, const)

Variables store data values. JavaScript has three types:

  • var: function-scoped, can be redeclared.
  • let: block-scoped, cannot be redeclared.
  • const: block-scoped, cannot be changed.
