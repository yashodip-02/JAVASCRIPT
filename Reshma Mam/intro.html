Introduction to JavaScript

JavaScript (JS) is a scripting language mainly used to add interactivity and dynamic behaviour in web pages.

History of JavaScript

  • 1995 – Created by Brendan Eich at Netscape (initially called LiveScript).
  • Later renamed to JavaScript for marketing reasons.
  • Standardised as ECMAScript (ES5, ES6/ES2015, etc.).

Client-Side vs Server-Side JavaScript

Client-Side JS
  • Runs inside the browser.
  • Used for form validation, DOM manipulation, animations.
  • Example: React, Vanilla JS in browser.
Server-Side JS
  • Runs on the server (e.g., Node.js).
  • Used for APIs, databases, business logic.
  • Example: Express.js with Node.

JavaScript Engines

A JavaScript engine is a program that executes JS code.

  • V8 – Chrome, Node.js
  • SpiderMonkey – Firefox
  • JavaScriptCore – Safari
  • Chakra (legacy) – older Microsoft Edge
